# CODEFORGE
<p> LeetCode-inspired coding platform with separate User and Admin portals for
problem solving, problem management, and progress tracking.</p>
<p> Integrated Judge0 API for automated code compilation and evaluation against custom and hidden test
cases across multiple programming languages.</p> 
<p> Implemented JWT authentication, Redis-based token invalidation, and sliding-window rate limiting to
ensure secure access and prevent API abuse.</p> 
<p> Integrated Gemini LLM to build an AI assistant within the platform, enabling users to ask programmingrelated questions and receive real-time guidance. </p>
